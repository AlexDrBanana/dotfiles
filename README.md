# Dotfiles

Personal shell configuration managed outside of the Prezto framework checkout.

Current managed links:

- `~/.zpreztorc` -> `~/dotfiles/zpreztorc`
- `~/.zshrc` -> `~/dotfiles/zshrc`
- `~/.zshenv` -> `~/dotfiles/zshenv`

The initial `zpreztorc`, `zshrc`, and `zshenv` files were copied from
`~/.zprezto/runcoms/` to preserve local settings that had previously been
committed inside the Prezto fork.
